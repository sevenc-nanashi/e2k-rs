# e2k-rs

[Patchethium氏のe2k](https://github.com/Patchethium/e2k)をRustに移植したものです。

## クレート一覧

- `./crates/e2k-rs`：メイン実装（Rust API）。 
- `./crates/e2k-js`：Wasm + JS API。
- `./crates/e2k-py`：PyO3によるPython API。
